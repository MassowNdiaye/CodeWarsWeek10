# JavaScript Coding Challenges

- Challenge 1 – String Ending Check

Checks whether a string ends with a given substring.

- Challenge 2 – Sum of Numbers

Calculates the sum of numbers in an array, supporting negative and decimal values. Returns `0` for an empty array.

- Challenge 3 – Unique In Order

Removes consecutive duplicate values from a string or array while preserving the original order.

- Challenge 4 – PIN Validation

Validates ATM PIN codes by checking length and ensuring all characters are digits.

- Challenge 5 – Disemvowel

Removes all vowels from a string using regular expressions.

- Challenge 6 – Filter List

Filters out values from a mixed array.

- Challenge 7 – Descending Order

Rearranges the digits of a number to create the highest possible value.

- Challenge 8 – Uppercase Conversion

Converts a string to uppercase.
